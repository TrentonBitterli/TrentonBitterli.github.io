Mini-Game Mayhem

Multi-Semester Project

Description

For one of my GIMM projects we were assigned to make an individual game using VR. Everything from the models to the game mechanics were made by me. I decided to make a mini-game platformer where the user will race against the clock in three different levels to test user speed and performance in a first person VR shooter. The game was made in Unity and it features a target practice, run n gun, and then a maze escape.

Skills Used

  • Unity Game Mechanics
  • C# Scripting
  • 3D Modeling in Blender and Maya'
  • Audio Engineering in Audacity'
  • VR Game Development
